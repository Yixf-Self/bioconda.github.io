perl-archive-tar

Home     http://metacpan.org/pod/Archive::Tar

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install perl-archive-tar

and update with::

   conda update perl-archive-tar
