Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/oases

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install oases

and update with::

   conda update oases



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/oases.
