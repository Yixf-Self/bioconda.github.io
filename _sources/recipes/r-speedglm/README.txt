r-speedglm
==========

|downloads|

Fitting linear models and generalized linear models to large data sets by updating algorithms.

======== ===========
Home     
Versions 0.3_1
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-speedglm
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-speedglm

and update with::

   conda update r-speedglm

A Docker container is available at https://quay.io/repository/biocontainers/r-speedglm.
