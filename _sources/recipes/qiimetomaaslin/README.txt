.. _`qiimetomaaslin`:

qiimetomaaslin
==============

|downloads|

Data munging script to change text Qiime OTU tables to pcl-formatted, maaslin-compatible text files

======== ===========
Home     https://huttenhower.sph.harvard.edu/maaslin
Versions 1.1.0
License  Unknown
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/qiimetomaaslin
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install qiimetomaaslin

and update with::

   conda update qiimetomaaslin



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/qiimetomaaslin.
