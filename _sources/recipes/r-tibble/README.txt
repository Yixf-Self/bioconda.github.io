r-tibble
========

|downloads|

Provides a 'tbl_df' class that offers better checking and printing capabilities than traditional data frames.

======== ===========
Home     https://github.com/hadley/tibble
Versions 1.2
License  MIT + file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-tibble
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-tibble

and update with::

   conda update r-tibble



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-tibble.
