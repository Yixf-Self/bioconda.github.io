hlama
=====

|downloads|

Simple matching of HTS samples based on HLA typing

======== ===========
Home     https://github.com/bihealth/hlama
Versions 0.3
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/hlama
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install hlama

and update with::

   conda update hlama



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/hlama.
