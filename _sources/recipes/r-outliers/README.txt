r-outliers
==========

|downloads|

A collection of some tests commonly used for identifying outliers.

======== ===========
Home     http://www.r-project.org, http://www.komsta.net/
Versions 0.14
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-outliers
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-outliers

and update with::

   conda update r-outliers



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-outliers.
