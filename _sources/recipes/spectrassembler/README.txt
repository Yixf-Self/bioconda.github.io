spectrassembler
===============

|downloads|

Tool (experimental) to compute layout from overlaps with spectral algorithm

======== ===========
Home     https://github.com/antrec/spectrassembler
Versions 0.0.1a1
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/spectrassembler
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install spectrassembler

and update with::

   conda update spectrassembler



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/spectrassembler.
