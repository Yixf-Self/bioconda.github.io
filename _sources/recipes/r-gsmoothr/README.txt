Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-gsmoothr

and update with::

   conda update r-gsmoothr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-gsmoothr.
