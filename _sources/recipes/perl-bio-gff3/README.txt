======== ===========

Versions 2.0

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install perl-bio-gff3

and update with::

   conda update perl-bio-gff3



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-bio-gff3.
