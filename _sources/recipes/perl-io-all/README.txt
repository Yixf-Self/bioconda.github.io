perl-io-all
===========

|downloads|

Combines all of the best Perl IO modules into a single nifty object oriented interface

======== ===========
Home     https://github.com/ingydotnet/io-all-pm
Versions 0.86
License  perl_5
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-io-all
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install perl-io-all

and update with::

   conda update perl-io-all



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-io-all.
