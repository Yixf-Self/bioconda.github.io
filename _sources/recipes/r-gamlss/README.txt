Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-gamlss

and update with::

   conda update r-gamlss

A Docker container is available at https://quay.io/repository/biocontainers/r-gamlss.
